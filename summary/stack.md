# 栈 (实现浏览器的前进和后退功能)

## 基础知识

```shell
    1. 栈是一种“操作受限”的线性表，只允许在一端插入和删除数据。
    2. 栈的产生是基于特定场景下的数据结构，而数组和链表暴露太多的操作接口，操作虽然灵活自由，
    　　但使用时不可控容易出错。
    3. 顺序栈: 用数组来实现
    　 链式栈: 用链表实现
    4. 入栈 push() 和 出栈 pop()
```

## 栈的应用

```shell
    1. 函数调用栈：　每个线程分配各自独立的“栈”空间, 存储在这个线程中函数调用时的临时变量。
    　　　　　　　　　每调用一个函数，就会将函数内临时变量作为一个栈帧入栈，当被调用函数执行完成，返回之后，
    　　　　　　　　　将这个函数对应的栈帧出栈。　用栈来实现主要还是跟变量的作用域有关
    
    2. 表达式求值
            3 + 5 * 8 - 4
            通过两个栈来实现的。其中一个保存操作数(数字)的栈，另一个是保存运算符的栈。从左向右遍历表达式，当遇到数字，我们就直接压入
            操作数栈；当遇到运算符，就与运算符栈的栈顶元素进行比较(如果运算符栈内没有元素就入栈)
            如果当前运算符比运算符栈顶元素的优先级高，就将当前运算符压入栈；如果当前运算符比运算符栈顶元素的优先级低或者相同，
            就从运算符栈中取栈顶运算符，从操作数栈的栈顶取 2 个操作数，然后进行计算，再把计算完的结果压入操作数栈，继续将
            当前的运算符与运算符栈顶元素进行比较。
            
    3. 检查表达式中的括号是否匹配
            括号包括圆括号(), 中括号[], 花括号{}
            {[{}]}
            定义一个栈用来保存未匹配左括号，从左到右依次遍历字符串。当取到到左括号时，则将其压入栈中；
            当取到到右括号时，从栈顶取出一个左括号，进行匹配，如果匹配成功则继续遍历剩下的字符串。
            遍历结束后，如果栈内还有左括号，则该表达式为非法格式．
```
